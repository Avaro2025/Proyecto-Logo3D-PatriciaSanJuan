# Manual de uso - Logo3D Patricia SanJuan

## Instalación
1. Clona el repositorio:  
   `git clone https://github.com/Avaro2025/Proyecto-Logo3D-PatriciaSanJuan.git`
2. Instala dependencias:  
   `npm install`
3. Ejecuta en local:  
   `npm run dev`

## Uso del panel de control
- Cambia propiedades PBR de cada elemento en tiempo real.
- Guarda y exporta presets desde el panel dat.GUI.

## Exportación
- El render final puede exportarse a WebGL, PNG, o vídeo según el flujo definido.
